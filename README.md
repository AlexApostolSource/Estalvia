# Estalvia

Estalvia is a project developed to help users manage their savings and finances efficiently. The application aims to provide tools for tracking expenses, setting savings goals, and gaining insights into personal financial habits.

## Features

- **Expense Tracking**: Easily log and categorize your expenses to understand where your money goes.
- **Savings Goals**: Set and monitor your savings objectives with clear progress indicators.
- **Financial Insights**: Visualize your spending and savings trends with intuitive charts and summaries.
- **User-Friendly Interface**: Simple and intuitive UI designed for easy navigation and quick data entry.
